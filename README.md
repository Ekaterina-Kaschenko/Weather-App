# Weather-App

## How-To
### Run application:

1. run:

  ```sh
  git clone https://github.com/Ekaterina-Kaschenko/weather-App
  ```
2. download and install Node.js from http://nodejs.org/ 
3. open folder with package.json in console
4. run:

  ```sh
  npm install
  ```
5. run:

  ```sh
  gulp
  ```
6. wait until page will be opened in your browser.
